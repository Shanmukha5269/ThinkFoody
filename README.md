# ThinkFoody

A beautiful recipe website with animated landing page using HTML, CSS, and Node.js.

## Features

- Recipe cards with hover effects
- Modern and clean UI

## Technologies Used

- HTML5
- CSS3 (with animations and transitions)
- Node.js 20+
- Express.js
